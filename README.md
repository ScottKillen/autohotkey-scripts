## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

Install [AutoHotKey](https://www.autohotkey.com/).

### Installation
 
1. Clone the repo
```sh
git clone https://github.com/ScottKillen/autohotkey-scripts.git
```
